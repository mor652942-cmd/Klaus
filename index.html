[ Sobre o projeto ]

Projeto Klaus


O Klaus é um projeto de sistema operacional que estou desenvolvendo para aprender mais sobre o funcionamento dos computadores. A ideia é evoluir o projeto aos poucos, sem pressa, aprendendo com os erros, testando novas ideias e colocando em prática tudo o que vou estudando. Este não é um projeto com a intenção de competir com sistemas já existentes. O objetivo é ganhar experiência, entender como um sistema operacional funciona desde o boot até o kernel e acompanhar a evolução do projeto a cada nova versão. Ainda há muito trabalho pela frente, mas faz parte da jornada. Cada linha de código escrita representa um novo aprendizado e um passo a mais no desenvolvimento do Klaus.
